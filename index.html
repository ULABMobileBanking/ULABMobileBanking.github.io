Premium Features

ULAB Mobile Banking is Providing the best possible features that you can ask for.
It’s incredibly easy and helpfull for ULAB family.

Registration

A new user can register for a new account.

User Login

A returning user can sign into their account using their identification number and security pin.

Send Money

Funds can be transferred between two accounts, provided the user has sufficient funds in their account.

Receive Money

A user can receive money from another user.

Display Data

The User can check to see the availability of funds (balance) in their account and other personal information.

Password Protected

All User Accounts & Transactions are protected by user provided PIN.

Deposit Money

The User can deposit funds to their account.

Withdraw Money

The User can choose to withdraw money from their account in the form of cash via an agent..
